# 📚 Boipoka Vibe

**Boipoka Vibe** is a simple and responsive book-related web application built with **React.js** and **Tailwind CSS**.
The project focuses on creating a clean UI and practicing fundamental React concepts with modern styling.

---

## 📌 Project Overview

Boipoka Vibe is designed for book lovers (📖 *“Boipoka” means bookworm*).
Users can explore books, view details, and interact with the interface dynamically.

This project was built to strengthen core front-end skills using:

* React component structure
* Props & state management
* Event handling
* Responsive UI design with Tailwind

---

## ✨ Features

*  Display book cards dynamically
*  View book details
*  Add to favorite / wishlist (if implemented)
*  Interactive UI with real-time updates
*  Fully responsive design
*  Clean and modern Tailwind styling

---

## 🛠️ Technologies Used

* **React.js** – Frontend JavaScript library
* **Tailwind CSS** – Utility-first CSS framework
* **JavaScript (ES6+)** – Application logic

## ⚙️ Installation & Setup

1. Clone the repository:

   ```bash
   git clone <your-repository-link>
   ```

2. Navigate to the project folder:

   ```bash
   cd boipoka-vibe
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Run the development server:

   ```bash
   npm run dev
   ```

5. Open in browser:

   ```
   http://localhost:5173
   ```
